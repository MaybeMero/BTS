A cosa serve bootstrap ?

Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice, Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, nonché plug-in JavaScript opzionali Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi

Cosa devi importare per creare una pagina web con bootstrap ?

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito ?

un sistema di computer collegati in rete attraverso Internet, che collaborano in maniera trasparente, sotto forma di sistema distribuito, per distribuire contenuti (specialmente contenuti multimediali di grandi dimensioni in termini di banda, come l'IPTV) agli utenti finali ed erogare servizi di streaming audio e video.

il ventaggio è che porta a tempi di caricamento più rapidi. Inoltre, la maggior parte dei CDN si assicurerà che una volta che un utente richiede un file da esso, verrà servito dal server più vicino a loro, il che porta anche a tempi di caricamento più rapidi.

A cosa serve questo attributo del tag meta?

Per garantire il rendering corretto e lo zoom tattile per tutti i dispositivi, aggiungi il metatag della finestra reattiva al tuo "head".

Che differenza c’è tra Container e Container Fluid ?

Il containerclasse fornisce un contenitore responsive a larghezza fissa, il container-fluidclasse fornisce un contenitore a larghezza intera , che copre l'intera larghezza della finestra

My Container

This part is inside a .container class.

The container class provides a responsive fixed width container.

My Container-Fluid

This part is inside a .container-fluid class.

The .container-fluid class provides a full width container, spanning the entire width of the viewport.
